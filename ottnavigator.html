Install OTT Navigator on Android Mobile

Step 1: Allow Unknown Sources (APK only)

  • Open Android Settings
  • Security / Privacy
  • Enable Install unknown apps

Step 2: Install App

  • Tap downloaded APK
  • Click Install
  • Open the app

Step 3: Grant Permissions

  • Storage access
  • Network access

Step 1: Open OTT Navigator

Step 2: Open Settings → Playlists

Step 3: Add Playlist → M3U URL

Step 4: Paste M3U URL

Step 5: Watch Live TV
